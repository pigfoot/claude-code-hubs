# Brand Logo Assets

## TrendLife Logo (2026)

**Source:** TrendLife 2026 brand logo design

- **File:** `trendlife-2026-logo-light.png`
- **Dimensions:** 2780 × 511 pixels (5.44:1 aspect ratio)
- **Git LFS:** Managed by Git LFS (requires `git lfs pull` after clone)

**Components:**

- Red circular icon with "t" symbol (left)
- "TRENDLIFE" text (black "TREND" + red "LIFE")
- PNG with transparency (RGBA)

**Specifications:**

- **Format:** PNG with transparency (RGBA)
- **Dimensions:** 2780 × 511 pixels (5.44:1 aspect ratio)
- **File Size:** Managed by Git LFS
- **DPI:** High resolution, suitable for all slide sizes

**Usage:**

- **Title/Cover Slides:** Used as reference image for AI generation (Gemini API only)
  - Logo integrated into design during generation
  - Not overlaid in corner
  - AI instructed to preserve logo exactly (trademark protection)
- **Content/Divider/End Slides:** Post-processing overlay in bottom-right corner
  - Size: 12-20% of slide width (varies by layout type)
  - Positioning: Layout-specific (see layout positioning table)

**Layout-Specific Positioning:**

| Layout  | Overlay Applied? | Position       | Size Ratio | Padding (H, V) | Opacity |
|---------|------------------|----------------|------------|----------------|---------|
| title   | NO (integrated)  | N/A            | N/A        | N/A            | N/A     |
| content | YES              | bottom-right   | 12%        | 40px, 30px     | 100%    |
| divider | YES              | bottom-right   | 15%        | 40px, 30px     | 100%    |
| end     | YES              | center-bottom  | 20%        | 0px, 50px      | 100%    |

**Guidelines:**

- Never distort aspect ratio
- Maintain minimum clear space (10% of logo width)
- Always use PNG with transparency
- Never recreate or modify logo (trademark protection)
- For title slides, logo is passed to AI as reference image (Gemini API only)
